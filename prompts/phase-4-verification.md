# Phase 4: Data Verification

Pick 2-3 examples from the tool's output and verify:

1. **Check live sources**
   - API calls to data providers
   - Compare timestamps
   - Verify numbers match

2. **Calculate real values**
   - Add taxes/fees the tool ignores
   - Check time estimates
   - Verify ROI calculations

3. **Document discrepancies**
   - Tool says X, reality is Y
   - Why the difference?
   - Impact on user decisions

Be ruthless. If the tool lies, call it out.
